# Guess the Number – Java Console Game

A simple interactive number guessing game written in Java.  
The computer picks a random number between 1 and 100, and you have to guess it.  
After each guess you'll get a "too high" or "too low" hint.

## Requirements
- Java 8 or higher (JDK)

## How to Compile and Run

### Using the convenience scripts (Linux/macOS or Windows)
- **Linux / macOS**: Open a terminal in the project folder and run  
  `./run.sh`
- **Windows**: Double-click `run.bat` or run it from Command Prompt

### Manual compilation and execution
1. Open a terminal/command prompt inside the project folder.
2. Compile:  
   `javac src/GuessNumberGame.java -d .`
3. Run:  
   `java GuessNumberGame`
